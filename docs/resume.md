## 🛠️ 代表的成果物（ポートフォリオ）

### [learning-ts-concepts](https://github.com/shadow-architect-dev/learning-ts-concepts)
AWS CDK および CDKTF を用いた、プロダクションレディな三層Webアーキテクチャテンプレート。

- **インフラ（AWS CDK / TypeScript）**:
  WAFv2 ➔ CloudFront ➔ ALB ➔ ECS on Fargate ➔ RDS Proxy ➔ Aurora Serverless v2 の構成。
  - **高可用性・耐負荷**: `stg`/`prod` では RDS Proxy とリードレプリカ (Reader 2台) をマルチAZ配置。
  - **コスト最適化**: `dev` 環境では Proxy をバイパス。夜間の ECS / Aurora 自動停止スケジュール、Nat Gateway 削減構成を実装。
  - **API制限対策**: Secrets Manager からのECSネイティブシークレット自動注入を実装し、タスクスケール時のAPIスロットリングを回避。
- **可観測性（Monitor as Code / CDKTF）**:
  CDKTF を用いて Datadog アラート定義を TypeScript でコード管理。S3 & DynamoDB リモートステート管理とローカル自動フォールバックを両立。
- **CI/CD & セキュリティゲート (GitHub Actions)**:
  Dockerイメージに対するHadolint (静的解析) および Trivy (脆弱性スキャン) によるデプロイブロックゲートを構築。
- **テスト自動化**: `aws-cdk-lib/assertions` によるリソース構成の単体テストを完備。
- **ガバナンス設計**: 実戦を想定した [マルチアカウント・セキュリティ・コスト管理方針](https://github.com/shadow-architect-dev/learning-ts-concepts/tree/main/docs/governance) をドキュメント化。

## 💡 自己PR

### 1. 経営・マネジメント視点とエンジニアリングのハイブリッド対応
CTO としてビジネス要件から技術戦略・財務コスト（FinOps）計画を策定するスキルと、リードエンジニアとして実際に Terraform/CDK のコードを書き、コンテナイメージを構築し、CI/CD や監視定義を実装するスキルの両面をシームレスに行き来できます。これにより、ビジネスのスピード感を損なわない最適なアーキテクチャ設計を提供します。

### 2. 生成AIをフル活用した圧倒的な自走力
新しいインフラ領域や言語のキャッチアップが必要な場面において、Gemini等の生成AIを「設計のペアプログラマ」および「高速調査エンジン」として極めて効率的に活用しています。ドキュメントの概念理解から、エラーログの即時デバッグ、品質の高いコードテンプレート生成までを AI との協調作業によって完遂し、他者の指示を仰ぐことなく完全自走でスピード感を持って課題を解決します。
